Jan Mackovic Marathon training sub 3 hours

Easy run- This is a recovery run where you will run short distance and with slow pace around 8:10-8:00 per mile to recover from other runs

Medium runs-This are medium range distance runs where you will run 10% slower then your marathon pace and that would be around 7:45

Long run-This is a run longer then half marathon and you will need to be 20% slower then your race pace and go to 10% if you can that would be around 8:20 per mile

Tempo runs- here you will run 5 or 6 miles with a pace faster than marathon pace around 6:45 to improve your speed

Interval workouts-Here you would run shorter distances for example 3x1600 where you will run 1 mile with your 5k race pace and then you would need to jog for 400m. If you are doing 400m intervals jog for 200m and if you are doing 200m or 100m take a 2 min brake walking
